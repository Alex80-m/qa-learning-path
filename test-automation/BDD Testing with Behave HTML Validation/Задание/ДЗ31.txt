Раздел 1. Формы и элементы ввода
1.1. Проверка всех обязательных полей

Написать тест, который проверяет, что:

#username имеет атрибут required

#password имеет атрибут required

#country имеет атрибут required

1.2. Проверка валидации логина по regex

Написать тест, который убеждается, что поле #username содержит pattern
^[A-Za-z0-9_\-]+$.

1.3. Проверка диапазона ползунка громкости

Проверить:

min = 0

max = 100

step = 5

значение по умолчанию = 70

1.4. Проверка выбора страны

Проверить, что:

в select есть 4 option’а

выбранная страна по умолчанию — "Россия"

есть optgroup "Европа" и "Америка"

1.5. Проверка email input с multiple

Написать тест, что:

поле #emails имеет атрибут multiple

тип = email

есть placeholder

1.6. Проверка кнопок формы

Написать тесты, что:

у кнопки #submit-btn стоит formaction="/save"

у #reset-btn тип reset

#upload-btn тип button и есть атрибут data-action="upload"

1.7. Проверка функциональности output (#calc-result)

Тест должен:

Изменить значение #age на 50

Изменить значение #volume на 80

Проверить, что в <output> появился текст с этими числами

1.8. Проверка file input

Тест проверяет:

multiple=True

accept содержит image/* и application/pdf

атрибут data-max-files="5"

Раздел 2. Таблицы
2.1. Количество колонок

Проверить, что в <thead><tr> ровно 3 th.

2.2. Проверка ролей пользователей в таблице

Проверить содержимое атрибута:

первая строка = "admin"

вторая строка = "editor"

2.3. Проверка summary и caption

Убедиться, что:

таблица имеет summary

в caption написано "Мини-таблица"

2.4. Проверка footer

Проверить, что:

<tfoot> существует

там написано: «Количество записей: 2»

Раздел 3. SVG / Canvas / Multимедиа
3.1. Проверка SVG логотипа

Написать тесты, которые проверяют:

наличие SVG #logo

наличие <title> и <desc>

наличие градиента <linearGradient id="g">

3.2. Проверка элементов canvas

Проверить:

canvas существует

width = 300

height = 120

3.3. Проверка video

Проверить:

video существует

есть атрибут controls

есть <track kind="captions">

3.4. Проверка audio

Проверить:

audio существует

тип управления controls

preload="none"

Раздел 4. Медиа, картинки, picture
4.1. Проверка picture-блока

Проверить:

<source> имеет media="(min-width:600px)"

внутри <img> есть alt

4.2. Проверка map & area

Написать тест:

есть карта map1

есть 2 area

первая area имеет title="Левая"

вторая area title="Правая"
