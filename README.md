QA Learning Path
Мой путь в QA: выполненные задания и проекты

Этот репозиторий содержит практические задания и проекты, выполненные в рамках обучения на курсах по специальности QA Engineer.  
Все материалы отражают мой личный опыт работы с ручным и автоматизированным тестированием и предназначены для демонстрации навыков при трудоустройстве.

Структура репозитория
- `manual-testing/` - Тест-кейсы, чек-листы, отчёты об ошибках
- `test-automation/` - UI-тесты на Selenium и Playwright (Python, pytest)
- `SQL/` - Запросы, анализ структуры БД, проверка данных
- `mobile-testing/` - Тестирование мобильных приложений с помощью Appium и эмуляторов Android
- `performance-testing/` - Нагрузочное тестирование в JMeter
- `QA Manual Final Work/` - Финальный проект: требования, тест-план, отчёты
- `git/` - Практика работы с Git и GitHub

Используемые технологии
- Языки: Python, SQL
- Автоматизация: Selenium WebDriver, Playwright, pytest, requests
- Инструменты: Postman, JMeter, Appium, Git, Allure, Docker
- Системы: Windows, Linux, Android (эмуляторы)
